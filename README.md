# Netflix Data Cleaning Project (Excel)

## Overview
This project demonstrates real-world data cleaning using Excel on the Netflix Movies and TV Shows dataset.

## Tasks Performed
- Identified and handled missing values
- Removed duplicate records
- Standardized text columns
- Converted text-based dates into YYYY-MM-DD format
- Validated numeric and categorical fields
- Added Data_Quality_Notes for documentation

## Tools Used
- Microsoft Excel

## Output Files
- Raw_Data.xlsx
- Cleaned_dataset.xlsx
- cleaned_dataset.csv
 ## Data Set
 -Netflix Movies & TV Shows (Kaggle): (https://www.kaggle.com/datasets/shivamb/netflix-shows)
